Excel to PDF Converter

This one needs the Windows app. Your own copy of Excel does the export, so print areas and calculated values land exactly where Excel puts them.

Download BenchPDF 1.0.0 for Windows Needs Microsoft Excel installed.

What comes through

  • Defined print areas, so you get the range you meant rather than the whole sheet.
  • Page breaks and scaling as set in the workbook.
  • Calculated values of formulas, as they stand when the file is opened.
  • Conditional formatting, cell styling and number formats.

Where it stops

A spreadsheet has no natural page size, so what you get depends entirely on the print settings saved in the workbook. If no print area is defined and the sheet is wide, Excel will split it across pages the way it would if you pressed print, which can be many pages. Setting the print area and scaling in Excel first gives you control over the result.

How to convert Excel to PDF

  • Set the print area and scaling in Excel if the sheet is wide.
  • Download and install BenchPDF, then open it.
  • Drop your .xlsx in, or pick Excel to PDF from the grid.
  • Download the result, or open the output folder from the app.

Will my print areas be respected?

Yes. Excel performs the export, so any print area, page break or scaling saved in the workbook is applied exactly as it would be if you printed from Excel yourself. That is also why it is worth setting those before converting a wide sheet.

Do formulas come out as values?

Yes. A PDF holds no formulas, so what appears is the calculated result as it stands when Excel opens the file. Conditional formatting driven by those values is applied too, so highlighted cells stay highlighted in the PDF.

Why is this not in the browser?

Excel's pagination is intricate, covering print areas, repeated header rows, scaling and page break preview. A browser cannot run that logic, so it would have to approximate it. Handing the file to your installed Excel means the PDF matches what Excel itself would print.

Do I need Excel installed?

Yes, because your copy of Excel does the work. If BenchPDF cannot find Excel on the PC, this conversion is greyed out with an explanation instead of falling back to a rougher layout that would not match your print settings.

Is my spreadsheet uploaded anywhere?

No. The app opens the workbook in the Excel already installed on your PC, exports it, and saves the PDF to your disk. There is no server in the conversion path at all.

Does it work on a Mac?

The app is Windows only, because it automates the Windows build of Excel. Excel for Mac has its own PDF export which behaves the same way, since Excel is doing the pagination in both cases.
