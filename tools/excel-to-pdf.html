Excel to PDF Converter

Uses your own installed Microsoft Excel to convert, so print areas and calculated values match exactly. Free Windows desktop app.

Download BenchPDF for Windows Free, no account. Requires Microsoft Excel installed.

Converting Excel to PDF accurately means using Excel's own print engine, since print areas, page breaks, and calculated formula values are specific to how Excel itself lays a sheet out. BenchPDF's desktop app drives your own installed copy of Excel directly, so the PDF matches exactly what Excel's own export would produce.

How to convert Excel to PDF

  • Download and install BenchPDF (free, no account).
  • Open the app and drop your .xlsx file, or choose Excel to PDF from the grid.
  • BenchPDF opens the spreadsheet in your installed Excel and exports it to PDF.
  • Download the result or open the output folder directly from the app.

Why isn't this available online, only as a desktop download?

Converting Excel to PDF accurately means using Excel's own print engine, since print areas, page breaks, and calculated formula values are specific to how Excel lays a sheet out. A browser cannot replicate that reliably. BenchPDF's desktop app drives your own installed copy of Excel directly, so the PDF matches what Excel's own export produces.

Is it safe? Does my spreadsheet get uploaded anywhere?

Yes, it is safe, and no, nothing is uploaded. The desktop app runs entirely on your own PC: it opens your spreadsheet in your installed copy of Excel, exports it to PDF, and saves the result to your disk. There is no server involved in the conversion at any point, and no account is required.

Do I need Microsoft Excel installed?

Yes. This conversion drives your own installed copy of Microsoft Excel to do the actual rendering, so Excel needs to already be on your PC. If it isn't detected, this specific conversion is greyed out in the app with an explanation, rather than failing partway through or approximating the layout badly.

Will print areas and formulas come out correctly?

Yes. Since the PDF is produced by Excel itself, defined print areas, page breaks, and the calculated values of any formulas all come out exactly as they would if you chose File then Export in Excel yourself. Conditional formatting and cell styling are preserved in the same way.

Is there a file size limit?

No artificial limit is set by BenchPDF. The practical limit is whatever your own copy of Excel can open, since the conversion is Excel doing the work. A small spreadsheet converts in a couple of seconds. A large workbook with many sheets takes Excel longer, exactly as a manual export would.

Does it work on Mac?

The BenchPDF desktop app is Windows only, since it automates the Windows version of Microsoft Excel directly. On a Mac, Excel has its own built-in Save as PDF export you can use instead, which produces a similarly faithful result since it is also Excel doing the rendering.
