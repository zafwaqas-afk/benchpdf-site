Web Page to PDF Converter

Paste a URL, get back a PDF, printed by a browser running on your own PC. Free Windows desktop app.

Download BenchPDF for Windows Free, no account. Windows desktop app.

To save a page as a PDF, something has to fetch that page, the same way your own browser would if you opened the address and pressed Print. BenchPDF does this with a local, headless browser running on your own PC, not a server elsewhere, and fetches only the address you give it.

How to convert a web page to PDF

  • Download and install BenchPDF (free, no account).
  • Open the app and paste the web address into the URL field.
  • BenchPDF fetches and prints the page to PDF using a local headless browser.
  • Download the result or open the output folder directly from the app.

Why isn't this available as a web tool?

A browser tab cannot fetch and render an arbitrary third-party web page on your behalf due to cross-origin browser security rules, the same restriction that stops any web page from silently loading another site's content. BenchPDF's desktop app works around this using its own local headless browser process, which a web page cannot do.

Does this upload anything to a server?

No. This is the one BenchPDF feature that makes a network request at all, and it is exactly the request needed to do the job: fetching the URL you give it, the same way your own browser would. Nothing else is contacted, and no copy of the page is kept anywhere except the PDF handed back to you.

Will it capture the page exactly as I see it?

In most cases, yes, since it uses a real browser engine to render the page before printing. Pages that require you to be logged in, or that load content only after you scroll or click, may not capture identically, since the headless browser sees the page fresh, without your existing session or manual interaction.

Is there a page length limit?

No artificial limit is set by BenchPDF. Very long pages produce a longer PDF and take a little more time to render, but there is no cap on page length, number of images, or file size, since everything runs on your own PC rather than a shared server with usage limits.

Can I convert a password-protected or private page?

Only if the headless browser can reach it without you being logged in, which most private pages will not allow. This tool fetches a page the way an anonymous visitor would, so content behind a login wall generally will not appear in the resulting PDF.

Does it work on Mac?

The BenchPDF desktop app is Windows only at present. On a Mac, most browsers offer their own Print to PDF option directly from the print dialog, which achieves a similar result for pages you can already see in your own browser.
