What comes through

  • Every page rendered pixel for pixel, the way a PDF viewer draws it.
  • Charts, diagrams and photographs exactly as they appear, with nothing rebuilt or guessed.
  • Colours and page proportions unchanged.
  • Your choice of PNG for sharp text, or JPG for a smaller file.

Where it stops

The output is a picture, so the text in it is no longer selectable or searchable. That is the point of this conversion rather than a fault, but if you need the words back afterwards, use PDF to Text on the original PDF instead. Very long documents at 300 DPI produce large downloads and use a lot of memory, since every pixel is generated on your own device.

How to convert PDF to images

  • Pick PNG or JPG and a resolution above, or leave the defaults.
  • Drop your PDF into the box, or choose a file.
  • Wait while each page renders.
  • Click Download to save a zip holding one image per page.

Will the images look exactly like the PDF?

Yes. This conversion does not try to extract or rebuild anything. It draws each page the same way a PDF viewer draws it on screen, then saves that as an image. Charts, embedded photographs, colours and positioning all come through as they appear in the original.

Which should I pick, PNG or JPG?

PNG keeps text edges sharp with no compression artefacts, which suits pages that are mostly type or line drawings. JPG makes a smaller file and suits photographic pages. The choice applies to every page of that PDF, and you can run the conversion again with the other setting.

What does the resolution setting change?

It sets how many pixels each page is drawn at. Use 96 DPI for viewing on screen, 150 for a good balance, and 300 when the image needs to hold up in print. Higher settings take longer to render and produce a noticeably bigger download.

Does my file get uploaded?

No. Each page is rendered inside your own browser using your device's own graphics, and the zip is assembled there too. Nothing about the PDF is transmitted. You can confirm it by watching your browser's network activity while a conversion runs.

Does it work on scanned PDFs?

Yes, and this is the conversion that handles them best. A scan is already a picture, so rendering it to PNG or JPG loses nothing at all. Conversions that read text, such as PDF to Text, are the ones that come back empty on a scan.

Does it work on a Mac?

Yes. This page runs in the browser, so macOS, Windows, Linux and ChromeOS all work. The separate BenchPDF desktop app is Windows only, but this conversion does not need it.
