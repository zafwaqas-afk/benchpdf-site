PDF to text converter

Pull the plain text out of a PDF, page breaks intact, without uploading it anywhere.

Convert online — no install Download desktop app

Try it online — same plain-text output, right in your browser.

BenchPDF extracts the plain text content of a PDF into a single .txt file, with a form-feed character between each page so the original page breaks stay identifiable. It's a straightforward conversion, useful for feeding a document into another tool, searching its content, or archiving it in a format that will still open in fifty years.

There's no per-document word or page cap tied to a hosted service's free tier here — it works the same way for a one-page letter as it does for a five-hundred-page report, because the only thing doing the work is your own PC.

And, as with the rest of BenchPDF, the PDF you're extracting text from is never uploaded anywhere to make that happen.
