Most online text-extraction tools upload your file to a server first. This tool reads the text layer of your PDF directly in your browser and writes it straight to a .txt file on your device. Nothing about the PDF or its contents is sent anywhere.

How to convert PDF to text

  • Drop your PDF into the box, or click to choose one from your device.
  • Wait a few seconds while your browser reads the text on each page.
  • Click Download to save the .txt file.
  • Open it in any text editor or word processor.

Is it safe to convert my PDF here?

Yes. The text is read out of your PDF using your own browser, and the resulting file is generated on your device. Nothing is uploaded to a server at any point. You can check this yourself by watching your browser's network activity while the conversion runs.

Will it keep the layout or paragraph breaks?

Line and paragraph breaks are preserved as closely as the PDF's own internal structure allows, and each page is separated by a form-feed character. Multi-column layouts and tables are flattened into a single reading order, since plain text has no concept of columns or page position.

Does it work on scanned PDFs?

No. A scanned page is a photograph of a document, with no underlying text for a browser to read, only pixels. This tool extracts existing text, it does not recognise text in an image (OCR). If a PDF is a scan, this tool will return an empty or near-empty result for that page.

Is there a file size or page limit?

No artificial limit is applied by this tool. The real constraint is your device's memory and processing power, since the extraction happens locally rather than on a server. A short PDF converts in under a second. A very long document takes a little longer, with no upload cap involved.

What format is the output file?

A plain .txt file, encoded as UTF-8, with pages separated by a form-feed character rather than any visible marker. It opens in any text editor, word processor, or code editor. There is no formatting, no fonts, and no images, only the words that were extracted from the PDF.

Does it work on Mac, or is it Windows only?

This online converter works on any device with a modern browser, including Mac, Windows, Linux, and Chromebooks, since it runs as JavaScript rather than a Windows program. BenchPDF also has a separate Windows desktop app with extra features, but you do not need it for this page.
