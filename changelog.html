Changelog

Every released version of BenchPDF, in order.

1.0.0 2026-07-15
  • Initial public release.
  • Local conversion hub: Word, Excel, and PowerPoint to and from PDF; images (including HEIC) to PDF; web page to PDF; PDF to images and to plain text.
  • In-browser PDF editor: click-to-edit text blocks over the original page, native table editing, export to a real, searchable PDF with untouched pages left byte-identical.
  • Office detection: Word/Excel/PowerPoint conversions are automatically disabled with a plain explanation on PCs without Microsoft Office installed, instead of failing.
  • Packaged as a self-contained Windows installer — no separate Python install required, no admin rights needed.
  • In-app About dialog with version, privacy statement, license, and the full list of bundled open-source components.
