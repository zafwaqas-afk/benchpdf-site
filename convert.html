This is a lighter engine than the desktop app. It reconstructs selectable text as PowerPoint slides using simple line-by-line positioning, and falls back to a page image when a PDF has no extractable text (a scan, for instance). It doesn't yet rebuild native tables, keep charts editable, or handle HEIC photos. For full fidelity — tables, chart-aware layout, Word/Excel/PowerPoint conversion via your own Office install, and in-place PDF editing — download the desktop app.
