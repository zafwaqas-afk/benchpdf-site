The promise

Your files never leave the device you are sitting at.

There is no upload step, no queue, and no account, because there is nowhere for the file to go. The desktop app converts on your own PC using your own installed copy of Office. The browser tools run as code inside your own browser. The one exception is saving a web page as a PDF, which has to fetch the address you give it, and nothing else.
