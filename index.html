Convert and edit PDFs without uploading them anywhere.

BenchPDF converts PDF, Word, Excel, PowerPoint, images, and web pages into each other, and lets you edit PDF text directly — all on your own Windows PC. There's no upload step, because there's nowhere for the file to go.

Convert online — no install Download for Windows

Online: PDF ↔ PowerPoint, PDF → Images/Text, Images → PDF — runs in your browser, nothing uploaded. Desktop app: every conversion (including Word, Excel, and PowerPoint via your own Office install) plus in-place PDF editing.

What BenchPDF does

Nothing uploaded

BenchPDF runs a small web page on your own PC (127.0.0.1) to give you a familiar drag-and-drop interface. That address means "this computer" — it isn't reachable from the internet, so there's no upload, no queue, and no file size limit tied to someone else's server.

Uses the Office you already have

Word, Excel, and PowerPoint conversions drive your own installed copy of those apps, so the output matches what Office itself would produce. If Office isn't installed, those conversions are clearly greyed out instead of failing or faking a result.

Edit a PDF in place

Click any text in a PDF to edit it directly in the browser view, then export a real, searchable PDF. Pages you didn't touch are exported byte-identical to the original.

Your files stay on your device

BenchPDF makes no network calls of its own. The one exception is “web page → PDF”, which — like opening a page in a browser — fetches the address you give it, and nothing else. There's no account to create, no cloud storage, and no analytics on this website either.

Every conversion

To PDF

Word → PDF Convert a Word document to PDF via Microsoft Word. Excel → PDF Convert a spreadsheet to PDF via Microsoft Excel. PowerPoint → PDF Convert a presentation to PDF via Microsoft PowerPoint. Images → PDF Merge one or more photos into a single PDF, in order. Web page → PDF Print a web page to PDF using a local headless browser.

From PDF

PDF → Edit & export Edit the text directly in the browser, then export a finished PDF. PDF → PowerPoint Editable slides. Tables, text, and images stay editable. PDF → Word Uses Word's own PDF import to rebuild an editable document. PDF → Images One PNG or JPG per page, at your chosen resolution. PDF → Text Plain text, one form-feed between pages.
